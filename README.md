# Relação Nominal de Servidores – MG

Este repositório gerencia **metadados, hash e publicação**
do dataset Relação Nominal de Servidores no dados.mg.

## Fonte oficial dos dados
https://dados.mg.gov.br/dataset/relacao_nominal_servidores

## O que este repositório faz
- Não armazena CSV
- Gera hash SHA256 dos arquivos do CKAN
- Publica/atualiza dataset via dpckan
- Serve base para GitHub Pages
