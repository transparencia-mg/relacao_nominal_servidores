Campo Descrição Tipo

Fonte dos Dados: Os dados são carregados da API CKAN do Portal de Dados Abertos de Minas Gerais. Cada arquivo CSV representa um mês específico (ex: dados_serv_202501.csv para Janeiro/2025).
Atualização Automática: Os dados são atualizados mensalmente com as novas relações de servidores.
